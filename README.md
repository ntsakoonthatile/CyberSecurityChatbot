Cybersecurity Awareness Chatbot

Overview
This project is a Cybersecurity Awareness Chatbot created using C# and WPF.

The chatbot helps users learn about cybersecurity topics such as:
- Password safety
- Privacy
- Phishing
- Online scams

The chatbot can also remember some user information and give different responses during conversations.

Features
- GUI interface
- Voice greeting
- Keyword recognition
- Random responses
- Memory and recall
- Sentiment detection
- Conversation flow

Technologies Used
- C#
- WPF
- Visual Studio 2022

How To Run
1. Open the project in Visual Studio.
2. Build the solution.
